mingtak.advertiser Installation
-------------------------------

To install mingtak.advertiser using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``mingtak.advertiser`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        mingtak.advertiser
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
